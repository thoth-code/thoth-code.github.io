정렬

파이썬에서 제공하는 강력한 정렬함수 : 객체.sort(), sorted(객체)

  1. 팀소트의 O(nlogn)의 아주 빠른 함수이다
  2. 객체.sort() : 해당 객체의 내부를 정렬한다. 리턴값은 없다
  3. sorted(객체) : 객체를 정렬해서 반환한다
  4. sort함수도 함수형 기능을 제공한다. 매개변수 key를 통해서 인데, 이것을 통해서 어떤 함수명을 전달해주면 이 함수의 실행 결과값을 기준으로 정렬한다. 물론 원래 객체 내부의 요소값은 변화하지 않고 말이다
    • 내가 이 기능을 다시 이용할지는 모르겠지만, key로 전달하는 함수의 리턴값이 튜플이라면, 인덱스가 우선인 것을 기준으로 정렬하고 이것으로 정렬이 불가능하면 그때 뒤의 인덱스로 기준을 옮겨가며 정렬한다
