Instance 추가, 수정, 삭제

Instance 추가

  • INSERT INTO 테이블이름 (칼럼1이름, 칼럼2이름)
    VALUES (칼럼1값, 칼럼2값);
  • 위 괄호에서 적어준 칼럼 이름 순서에 맞게 그 칼럼에 들어갈 값들을 아래 괄호에 적어주면 된다
  • 또한 값들을 칼럼 조건에 맞게(도메인, 제약조건, NOT NULL같은 것들에 위배되지 않게) 적어주면 된다
  • 칼럼이름 안적어줘도 칼럼에 들어갈 값들을 테이블 생성시의 칼럼 순서에 맞게 써주면 별 문제 없드라

Instance 수정

  • UPDATE 테이블이름
    칼럼1이름 = 칼럼1값
    칼럼2이름 = 칼럼2값
    WHERE 조건;
  • 인스턴스를 선택하는것은 무조건 저 WHERE절을 이용하므로 조건을 걸어주지 않으면 모든 인스턴스가 수정된다.

Instance 삭제

  • DROP FROM 테이블이름 WHERE 조건;
  • 인스턴스를 선택하는 것은 WHERE절을 이용하므로 조건을 걸어주지 않으면 모든 인스턴스가 삭제된다.
