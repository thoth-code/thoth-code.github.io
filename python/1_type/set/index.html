집합
Code
Concept

선언

  • a = {1, 2, 3, 4}
  • 키:벨류 쌍이 아닌 요소만 중괄호에 묶으면 집합이 된다
  • 집합의 요소도 불변객체여야 한다. 리스트를 요소로 가질 수 없다 이말이다

생성자

  • a = set()
  • a = set(iterable)
  • 인자를 안넣으면 빈 집합이 만들어진다
  • 반복가능한 객체를 넣으면 그 객체의 내용이 집합화되어 반환된다

교집합

  • a & b
  • a.intersection(b)
  • 두 잡합의 교집합을 구하는 연산이다

합집합

  • a | b
  • a.union(b)
  • 두 집합의 합집합을 구하는 연산이다

차집합

  • a - b
  • a.difference(b)
  • 두 집합의 차집합을 구하는 연산이다
  • 교집합과 합집합과는 다르게 차집합은 순서에 신경써야된다(a - b, b - a는 다르다)
