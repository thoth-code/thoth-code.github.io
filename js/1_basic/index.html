웹브라우져에서 보기

  • option + b
  • 맥 vscode에서 옵션키 + b를 누르면 html파일이 기본 웹 브라우져에서 실행된다
  • html파일이 아닌 js파일에서 실행시키면 그냥 코드가 화면에 출력된다

html파일에 자바스크립트 추가하기

  • <script>코드</script>
  • 저 태그 사이에 자바스크립트 코드를 넣으면 실행되게 된다
  • 우리나라에서는 보통 head태그 안에 집어넣는댄다

코드 외부파일로 분리하기

  • <script src="파일경로"></script>
  • 이렇게 자바스크립트 코드를 html안에 넣지 않고 외부로 분리해서 깔끔하게 관리할 수도 있다.
