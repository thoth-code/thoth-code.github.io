단순한 화면 출력

  • document.write("aloha");
  • 화면에 aloha가 출력되게 된다

CSS의 .class를 이용하기

  • /*html파일*/

    <div class="c_name_"></div>
  • /*js파일*/

    var arg = document.
    querySelector('.c_name_');

    arg.textContent = 'aloha';
  • document.querySelector로 CSS클래스를 선택하고
  • arg.textContent = 로 안에 들어갈 문자열을 적는다
