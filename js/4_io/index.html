화면 입출력

화면 출력

  • document.write("aloha");
  • 화면에 aloha가 출력되게 된다

팝업창 띄워서 입력받기

  • var a = prompt("질문", "기본값");
  • 화면에 팝업창으로 박스가 하나 뜨면서 질문을 출력하고 그 아래 입력창에 기본값이 들어있는 경태로 사용자의 입력을 기다리게 된다

팝업창으로 확인/취소 입력받기

  • confirm("질문");
  • 화면에 팝업창으로 박스가 하나 뜨면서 질문을 출력하고 확인/취소 버튼으로 사용자가 선택할 수 있게 하는 놈이다
  • 사용자가 확인을 누르면 true값을 반환하고 취소를 누르면 false값을 반환한다

경고창 띄우기

  • alert("error!!!");
  • '닫기'의 선택지밖에 없는 경고성 멘트를 팝업으로 띄울 때 사용한다
